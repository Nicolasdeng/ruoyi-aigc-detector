# AI 图片检测配置说明

## 配置文件位置

在 `ruoyi-admin/src/main/resources/application.yml` 中添加以下配置：

## 基础配置

```yaml
# AI检测配置
ai:
  detection:
    # 全局超时时间（秒），默认60秒
    timeout: 60
    
    # Hugging Face API配置
    huggingface:
      # API Token（可选，不配置会使用免费额度）
      token: 
      
    # HTTP代理配置（用于访问国外API）
    proxy:
      # 是否启用代理
      enabled: false
      # 代理服务器地址
      host: 127.0.0.1
      # 代理服务器端口
      port: 7890
```

## 配置说明

### 1. 超时时间配置

- **timeout**: 设置API调用的超时时间
  - 默认值：60秒
  - 建议值：30-120秒
  - 如果网络较慢，可以适当增加

### 2. Hugging Face Token配置

- **huggingface.token**: Hugging Face API的访问令牌
  - 不配置时使用免费额度（有限制）
  - 配置后可获得更高的调用限额
  - 获取方式：访问 https://huggingface.co/settings/tokens

### 3. 代理配置（重要）

由于Hugging Face等API服务器在国外，访问可能需要代理：

#### 方式1：使用Clash等代理工具（推荐）

```yaml
ai:
  detection:
    proxy:
      enabled: true
      host: 127.0.0.1
      port: 7890  # Clash默认端口
```

#### 方式2：使用v2ray等代理

```yaml
ai:
  detection:
    proxy:
      enabled: true
      host: 127.0.0.1
      port: 10808  # v2ray常用端口
```

#### 方式3：不使用代理

```yaml
ai:
  detection:
    proxy:
      enabled: false
```

**注意**：不使用代理时，Hugging Face API可能无法访问，但本地特征检测器仍可正常工作。

## 检测器说明

系统包含多个检测器，它们会并发执行：

### 1. 本地特征检测器
- **权重**: 20%
- **特点**: 
  - 无需网络，响应速度快
  - 基于文件名、尺寸、格式等特征分析
  - 准确率较高，尤其对AI生成图特征明显的图片
- **始终可用**

### 2. Hugging Face AI检测器
- **权重**: 30%
- **特点**:
  - 基于深度学习模型
  - 准确率高
  - 需要网络访问
- **需要代理或直连国外网络**

### 3. 其他检测器
- 根据系统配置自动加载
- 权重和功能各异

## 检测流程

1. 所有检测器并发执行（最多等待10秒）
2. 10秒后收集已完成的检测结果
3. 至少需要1个检测器成功才能返回结果
4. 系统会自动聚合所有结果，计算最终置信度

## 优化建议

### 1. 提高检测速度
- 降低timeout值（如30秒）
- 确保代理稳定高速

### 2. 提高检测准确率
- 配置Hugging Face Token
- 启用代理访问外部API
- 使用高质量代理服务

### 3. 仅使用本地检测
- 设置 `proxy.enabled: false`
- 本地检测器优化后准确率已显著提升
- 响应速度最快，无需网络

## 故障排查

### 问题1：所有检测引擎均失败

**可能原因**：
- 本地特征检测器出错（文件路径问题）
- 网络完全不可用

**解决方法**：
- 检查文件上传路径配置
- 查看日志确认具体错误

### 问题2：Hugging Face API超时

**可能原因**：
- 未配置代理
- 代理不可用
- 网络不稳定

**解决方法**：
1. 启用代理配置
2. 检查代理工具是否运行
3. 增加timeout值
4. 或者关闭代理，仅使用本地检测

### 问题3：检测速度慢

**可能原因**：
- timeout设置过大
- 等待失败的API响应

**解决方法**：
- 降低timeout值（建议30-60秒）
- 系统会在10秒后返回已完成的结果

## 检测准确率说明

### 本地特征检测器（已优化）

**优化内容**：
1. 扩展AI尺寸检测范围（256-4096）
2. 增加对齐检测（64/128倍数）
3. 支持中文AI工具名称（豆包、通义、文心等）
4. 提高基准评分（20%-95%）
5. 多高权重特征加成
6. 降低判定阈值（50%）

**适用场景**：
- AI生成特征明显的图片（准确率80%+）
- 文件名包含AI关键词
- 标准AI尺寸（512/768/1024/2048等）
- PNG格式且带透明通道

**局限性**：
- 对特征不明显的图片准确率降低
- 无法分析图片内容本身
- 需配合其他检测器使用
